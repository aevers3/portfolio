Train Scheduling App

About

This application is primarily an exploration into persisting server data, making use of Google Firebase. It allows users (probably some train administrator), to add trains to the onscreen schedule.

The train arrivals are also tracked over time using the given arrival intervals and first train arrival time. All time calculations are performed using Moment.js.

Technologies used in this project:

HTML/CSS Bootstrap Javascript jQuery Firebase Moment.js

Planet Trivia

About

This is a multiple choice quiz with a 15-second time limit on each question. There are 10 questions, and the user is presented with his or her score upon completion of the quiz.

The questions and answers are stored in a JS object, so swapping in new questions or quizzes can be done simply.
